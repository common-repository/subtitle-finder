This plugin can search Online in subscene.com and give to your users Subtitle of movies

== Description ==
by this plugin you can make a online subtitle Finder.
you can make a search form by [subtitle_finder_searchform] Shortcode.
and you can list subtitle of a plugin by [subtitle_finder movie='test'] Shortcoe in your posts and pages .

== Installation ==

1. Upload plugin 
1. Active plugin from plugins menu
1. go to settings and Subtitle Finder and set your language and others
